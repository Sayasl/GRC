# ISO 27001 Governance, Risk & Compliance (GRC) Project

A practical Governance, Risk & Compliance (GRC) project designed for small organizations implementing an Information Security Management System (ISMS) aligned with **ISO/IEC 27001** principles.

This repository provides a structured approach to managing information security risks through documented policies, risk tracking, compliance monitoring, and reporting.

---

## Project Overview

This project demonstrates how a small company can establish foundational GRC processes and maintain security governance documentation.

The project includes:

- Information Security Risk Register
- Risk Assessment Methodology
- Risk Register Checklist
- Security Policy
- Dashboard for Risk Monitoring
- Compliance Reporting

---

## Objectives

- Establish a repeatable information security risk management process
- Identify and assess business and cybersecurity risks
- Track remediation actions and ownership
- Measure security control effectiveness
- Support ISO 27001 readiness and audit preparation
- Improve visibility through dashboard reporting
